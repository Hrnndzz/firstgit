Configuring Git

  1. Go to VS code and click Terminal. arrow the new terminal
  2. Check if git is already installed by typing git --version in the command line
  3. Configure git in your local computer
    • Set your Username
      git config --global user.name "your name"
    • Set your Email address
      git config --global user.email "email"
  4. To initialize the project directory type git init
